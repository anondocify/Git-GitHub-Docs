# Release Files & Tools

This file contains the release binaries and installers included in this repository.

## 1. Git for Windows (Includes Git Bash)
**Git Bash** is the command-line interface for Git on Windows.

### Offline Installers (Available in this folder)
- **[Git-2.52.0-64-bit.exe](./Git-2.52.0-64-bit.exe)** (Standard 64-bit Windows)
- **[Git-2.52.0-arm64.exe](./Git-2.52.0-arm64.exe)** (ARM64 Windows)

### Official Download
- **Link**: [https://git-scm.com/download/win](https://git-scm.com/download/win)
- **Installation Note**: During installation, ensure you keep the checkbox **"Git Bash Here"** selected.

## 2. GitHub Desktop
**GitHub Desktop** is a graphical user interface (GUI) for Git.

### Offline Installer
- **[GitHubDesktopSetup-x64.exe](./GitHubDesktopSetup-x64.exe)** (Windows 64-bit)

### Official Download
- **Link**: [https://desktop.github.com/](https://desktop.github.com/)

## 3. Visual Studio Code (VS Code)
A popular code editor with built-in Git support.

### Offline Installers
- **Windows**: **[VSCodeUserSetup-x64-1.107.1.exe](./VSCodeUserSetup-x64-1.107.1.exe)**
- **Linux (Debian/Ubuntu)**: **[code_1.105.1-1760482543_amd64.deb](./code_1.105.1-1760482543_amd64.deb)**

### Official Download
- **Link**: [https://code.visualstudio.com/download](https://code.visualstudio.com/download)

## 4. Other Platforms

### Git for Linux
